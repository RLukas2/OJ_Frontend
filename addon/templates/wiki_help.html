{{ _('Judge Status') }}

  • Waiting Your submission is currently inqueue
  • Fetched Your submission has been fetched by the judger
  • Compiling Your submission is being compiled
  • Judging Your submission is being judged
  • Accepted Your code passed all the tests
  • Wrong Answer Your code's output is not correct
  • Time Limit Exceeded Your code doesn't fit the TL
  • Memory Limit Exceeded Your code doesn't fit the ML
  • Runtime Error Your code return a value different from 0 (array bounds, divide by zero,...)
  • Compile Error Your code couldn't compile
  • System Error Something is wrong with the judger/server (please let us know by posting a blog/asking a question)
  • Canceled Your submission has been canceled
  • Unknown Error Strange things happened
  • Ignored Your submission was ignored
If your submission got canceled/ignored, probably the admin had found out that there's another submission has the same code like yours

{{ _('Contest') }}

In OI mode, the last submission will be counted as the result.

In OI mode, highest total score is ranked first. Same score will be same rank.

In ACM/ICPC mode, passed code will be counted as the result.

In ACM/ICPC, the highest total number of problems solved is ranked first. If they're same, penalty will decide.
